# Sistema de Inventario en C

Proyecto desarrollado para la materia Programación I.

## Funcionalidades

- Ingreso de productos (nombre y precio)
- Cálculo del total del inventario
- Cálculo del promedio
- Producto más caro
- Producto más barato
- Búsqueda de productos por nombre

## Tecnologías

- Lenguaje C
- Uso de arreglos
- Funciones
- Archivos header (.h)

## Cómo ejecutar
*lo que esta entre "..." es lo que se debe escribir*

- Paso 1:
  - En la carpeta descargada, escribir "cmd" en la barra de dirreccion.
- Paso 2:
  - En el programa cmd escribir "dir" para verificar que esten los archivos.
- Paso 3:
  - Escribir "gcc main.c inventario.c" para compilar el programa.
- Paso 4:
  - Escribir "inventario" para ejecutar el programa.
